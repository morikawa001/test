取扱説明

このWebアプリは、臨床研究研修会のマスターCSV管理、研修会情報の入力・更新、日程の自動計算、 保存前確認、更新CSVの出力、テンプレートファイルへの一括差し込みを行うための管理ツールです。[file:1]

1. このアプリでできること

  • マスターCSVを読み込んで既存データを表示・編集できます。[file:1]
  • 新規DB作成から空のマスターを開始し、1件目から登録できます。[file:1]
  • 開催日や開始時刻から、曜日、入室時刻、所要時間、各締切日、メール用日付などを自動計算します。[file:1]
  • 保存前に確認画面で内容・注意事項・上書き/新規追加の状態を確認できます。[file:1]
  • 保存後は更新済みCSVをダウンロードできます。[file:1]
  • docx / pptx / xlsx テンプレートへ、選択Noの内容を一括差し込み出力できます。[file:1]

2. 画面の見方

  • Quick Operation では、テーマ切替、マスターCSV読込、新規DB作成、確認画面への移動、No選択を行います。[file:1]
  • Annual Plan Parameter では、年度件数、開催済み件数、直近30日件数、追加予定件数をリング表示します。[file:1]
  • Deadline Alert では、7日以内に期限が迫る起案や締切関連タスクを表示します。[file:1]
  • Entry Console では、研修会の基本情報・Zoom情報・説明文などを入力します。[file:1]
  • Computed Schedule では、自動計算された起案日、HP公開日、締切、チェック状態を確認できます。[file:1]
  • Master Table では、現在保持しているマスター一覧を確認できます。同じNoは1行だけ保持され、保存時は上書き更新されます。[file:1]
  • Commit Check では、保存前の確認内容と注意事項を一覧表示します。[file:1]
  • Export Result では、保存後のCSV出力状態を表示します。[file:1]
  • Template Injection では、テンプレートファイルを複数選択し、選択Noの値で一括差し込みできます。[file:1]

3. 基本的な使い方

  1. まず マスターCSV の「csv選択」ボタンから既存CSVを読み込むか、新規DB作成 を押して空のマスターを作成します。[file:1]
  2. 既存データを編集したい場合は、No選択欄から対象Noを選び、選択Noを反映 を押します。[file:1]
  3. 新規追加したい場合は、必要に応じて 直近行を複製 を使い、前回入力内容をベースに編集します。[file:1]
  4. Entry Console で各項目を入力または修正します。[file:1]
  5. 必要に応じて右側のチェック欄で、起案1 / HP / 起案2 / 起案3 の実施済み状態を設定します。[file:1]
  6. 保存する または 確認画面へ進む を押して内容を確認します。[file:1]
  7. 問題がなければ この内容で保存 を押します。[file:1]
  8. 最後に 更新CSVをダウンロード を押して、最新のCSVを保存します。[file:1]

4. CSV読込と新規DB作成

マスターCSVを読み込むと、CSV内の有効行を取り込み、No順で整理したうえで一覧・統計・期限アラート・差し込み対象Noが更新されます。[file:1] 有効データがある場合は、最後の行をもとに次回入力候補が自動で補完されます。[file:1]

新規DB作成では、既定ヘッダーを持つ空のマスターを初期化し、Noは1から開始されます。[file:1] この状態でもそのまま入力・保存・CSVダウンロードが可能です。[file:1]

5. 入力項目と自動計算

開催日、開始時刻、終了時刻、講義開始時刻、事前打合せ、質疑応答締切、講師名、タイトル、対象、開催方法、共催、Zoom情報、HP URL、目的、各種紹介文などを入力できます。[file:1]

開催日を入力すると曜日、メール用開催日、各種締切日、資料DL締切、講義資料締切が自動計算されます。[file:1] 開始・終了時刻からは入室時刻と所要時間が自動計算されます。[file:1]

対象区分に応じて対象詳細が自動設定されます。研究者の場合は「（医師・歯科医師等）」、研究支援者の場合は「（薬剤師・看護師等）」になります。[file:1] 共催が「企画人材班」の場合は、共催3やスライド用ヘッダー文言も連動して反映されます。[file:1]

6. 起案日・締切日の考え方

  • 起案1は開催35日前です。[file:1]
  • HP公開は開催28日前です。[file:1]
  • 起案2は開催7日前です。[file:1]
  • 起案3は開催翌営業日基準で自動計算されます。[file:1]
  • 院外締切は開催8日前の12:00です。[file:1]
  • 資料DL締切は開催翌営業日基準で表示されます。[file:1]
  • 講義資料締切は開催7日前です。[file:1]

土日および日本の主要祝日は営業日計算から除外されるように処理されています。[file:1]

7. 保存の仕組み

保存時は、入力中のNoを基準に、既存Noがあれば上書き、存在しなければ新規追加として扱われます。[file:1] そのため、同じNoが複数行に増えることはなく、マスター内では同一Noは1件として保持されます。[file:1]

保存対象には、基本情報だけでなく、起案1 / HP / 起案2 / 起案3 のチェック状態も含まれます。[file:1]

8. 保存前確認

確認画面では、保存予定件数、要確認件数、保存状態、新規追加か上書きか、自動計算項目、起案チェック状態を一覧表示します。[file:1]

No未入力、Noが整数でない、開催日未入力、タイトル未入力、講師名未入力、開始/終了時間未入力、終了時刻が開始時刻以前、Zoom URLやHP URLの形式不正、謝金が数値でない場合は注意事項として表示されます。[file:1]

9. CSV出力

保存後または編集中の状態から、更新CSVをダウンロードできます。[file:1] 出力ファイル名は「開催日_タイトル.csv」の形式で自動作成されます。[file:1]

CSVはUTF-8 BOM付きで出力されるため、日本語環境でも扱いやすい形式になっています。[file:1]

10. テンプレート一括差し込み

Template Injection では、docx / pptx / xlsx ファイルを複数まとめて選択し、差し込み対象Noを選ぶことで一括出力できます。[file:1]

差し込み時は、CSVヘッダー名に加えて、DATE_1、DATE_2、TITLE、SPEAKER、NO、PASSCODE、ZOOM_URL、WEBINER_ID などの別名キーにも対応しています。[file:1] そのため、テンプレート側で置換文字列をそろえておけば、Word・PowerPoint・Excelに対してまとめて差し込み可能です。[file:1]

出力ファイル名は「No{番号}_{タイトル}_{元ファイル名}」の形式になります。[file:1]

11. テーマ切替

「表示テーマ切替」ボタンで、ダークテーマとライトテーマを切り替えられます。[file:1] ダークテーマはグリーン端末調、ライトテーマは Notion 風の明るい業務UIとして設計されています。[file:1]

12. 使い方の例

  1. 「csv選択」から既存マスターCSVを読み込みます。[file:1]
  2. Noを選択して既存研修会を反映し、開催日や講師名だけ修正します。[file:1]
  3. 自動計算された締切や起案日を右側で確認します。[file:1]
  4. 「確認画面へ進む」で内容を確認し、「この内容で保存」を押します。[file:1]
  5. 「更新CSVをダウンロード」で新しいマスターCSVを保存します。[file:1]
  6. 必要ならテンプレートファイルを選択し、同じNoで差し込み資料も出力します。[file:1]

13. 注意事項

  • CSVを差し替えたあとは、必ず最新の更新CSVをダウンロードして保管してください。[file:1]
  • 同じNoで保存した場合は上書き更新になります。[file:1]
  • テンプレート差し込みは、docx / pptx / xlsx のみ対応です。[file:1]
  • URL項目は http または https で始まる形式で入力してください。[file:1]
  • 開催日や時刻を変更した場合は、自動計算欄と確認画面を必ず確認してください。[file:1]
